# Product Requirements Document (PRD): Интерактивный ИИ‑канвас

## 1. Введение и Видение

**Продукт:** Интерактивный ИИ‑канвас.

**Видение:** Создать интуитивно понятный инструмент, который позволяет пользователям преобразовывать неструктурированную информацию (тексты, идеи, документы) в структурированные, визуально привлекательные карты знаний. Пользователи могут итеративно дорабатывать эти карты в диалоге с ИИ, что делает процесс исследования и организации информации быстрым и эффективным.

## 2. Целевая Аудитория

*   **Аналитики и исследователи:** для быстрого структурирования информации из больших объемов текста.
*   **Студенты и преподаватели:** для создания наглядных конспектов и учебных материалов.
*   **Проектные менеджеры и стратеги:** для визуализации планов, идей и сложных концепций.
*   **Креативные специалисты:** для мозговых штурмов и организации идей.

## 3. Пользовательские Сценарии и Функции

### 3.1. Основные Функции (MVP)

| ID | Пользовательский сценарий | Описание функции |
| :--- | :--- | :--- |
| F-01 | **Ввод информации** | Пользователь может ввести или вставить произвольный текст (идеи, тезисы, статью) в специальное поле. |
| F-02 | **Автоматическая генерация графа** | После ввода текста ИИ анализирует его и автоматически создает визуальный граф, состоящий из нод (ключевые концепции) и связей (отношения между ними). |
| F-03 | **Визуализация графа** | Сгенерированный граф отображается на бесконечном канвасе. Система автоматически расставляет ноды для обеспечения читаемости и эстетики. |
| F-04 | **Интерактивный канвас** | Пользователь может панорамировать (pan) и масштабировать (zoom) канвас для удобной навигации по графу. |
| F-05 | **Диалоговое редактирование** | Пользователь может взаимодействовать с ИИ через чат для доработки графа. Примеры команд: "Объедини эти две ноды", "Добавь больше деталей к этой теме", "Упрости эту ветку". |
| F-06 | **Ручное редактирование** | Пользователь может вручную перемещать ноды, чтобы настроить их расположение. |

### 3.2. Функции для будущих версий

*   **Расширенное ручное редактирование:** Добавление, удаление и переименование нод и связей вручную.
*   **Экспорт графа:** Возможность сохранять граф в форматах PNG, SVG или PDF.
*   **Шаблоны:** Предустановленные шаблоны для разных типов задач (например, SWOT-анализ, дорожная карта проекта).
*   **Совместная работа:** Возможность нескольким пользователям одновременно работать над одним графом.

## 4. Критерии Успеха

*   **Качество генерации:** 90% сгенерированных графов должны быть логически связными и требовать минимальных ручных правок.
*   **Скорость:** Время от ввода текста до отображения графа не должно превышать 10 секунд для текста объемом до 5000 символов.
*   **Вовлеченность пользователей:** 70% новых пользователей возвращаются к продукту как минимум один раз в течение первой недели.
*   **Удовлетворенность пользователей:** Оценка по опросам не ниже 4.5/5.
