# Decision Log

This file serves as persistent project memory for non-trivial changes (architecture, behavior, workflow, or conventions).

## Format
*   **Context:** What was the problem or situation?
*   **Decision:** What was decided?
*   **Rationale:** Why was this decision made?
*   **Trade-offs:** What are the downsides or alternatives considered?
